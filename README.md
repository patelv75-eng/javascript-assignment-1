Q1. What is innerHTML?

 innerHTML is a property of the Document Object Model (DOM) in web development that allows JavaScript to read or update the HTML content inside a given element.


Q2. What is DOM?

Document Object Model
a programming interface that represents web pages as a tree of objects, allowing scripts to dynamically interact with and modify the structure, style, and content of the document.
